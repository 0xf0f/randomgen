randomgen.mt19937.MT19937.seed¶

MT19937.seed(seed=None)¶

Seed the generator.

Parameters

seed ({None, int, array_like}, optional) – Random seed initializing the pseudo-random number generator. Can be an integer in [0, 2**32-1], array of integers in [0, 2**32-1] or None (the default). If seed is None, then MT19937 will try to read entropy from /dev/urandom (or the Windows analog) if available to produce a 32-bit seed. If unavailable, a 32-bit hash of the time and process ID is used.

Raises

ValueError – If seed values are out of range for the PRNG.
