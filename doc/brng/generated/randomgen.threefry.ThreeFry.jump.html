randomgen.threefry.ThreeFry.jump¶

ThreeFry.jump(iter=1)¶

Jumps the state of the random number generator as-if 2**128 random numbers have been generated.

Parameters:iter (integer, positive) – Number of times to jump the state of the rng.
Returns:self – PRNG jumped iter times
Return type:ThreeFry

Notes

Jumping the rng state resets any pre-computed random numbers. This is required to ensure exact reproducibility.
